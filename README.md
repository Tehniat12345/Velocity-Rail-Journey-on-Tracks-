🚆 Velocity Rail | Journey on Tracks

Velocity Rail is a modern and stylish train landing page built using pure HTML and CSS. The project features a futuristic railway-inspired design with clean layouts, bold typography, smooth hover effects, and a professional user interface.

The landing page is designed to showcase a premium train travel experience with attractive visuals and responsive structure.

✨ Features
Modern Railway Inspired UI
Fully Responsive Design
Smooth Hover Animations
Clean and Structured Layout
Professional Typography
HTML5 & CSS3 Only
🛠 Technologies Used
HTML5
CSS3
Google Fonts
📌 Purpose

This project was created to practice frontend web development skills and improve creativity in landing page design using only HTML and CSS.
